# API contracts

Este diretório documenta os contratos REST solicitados no escopo original.

No Next.js App Router, as rotas executáveis ficam em `/app/api/*/route.ts`.

Endpoints implementados:

- `POST /api/generate`
- `POST /api/export`
- `POST /api/ai-scenario`
- `GET /api/dashboard`
- `GET /api/logs`
- `GET /api/mitre`
